# MINESWEEPER
Basic Minesweeper game made with HTML and Javascript.  
Made for fun and to learn more of HTML and JS.  

<details>
  <summary>In case you don't know how to play Minesweeper:</summary>
  <p>
  Click on a square to reveal it. If the square has a bomb, you lose. <br>
  If it doesn't, it reaveals a number. The number tells you how many mines are <br>
  surrounding that square. Click on "Mode: Reveal" to change it to "Mode: Flag" <br>
  You place a flag when you are sure there is a mine in that spot.</p>
</details>
